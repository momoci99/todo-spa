# Simple todo list (local)

이것 저것 테스트 해보는 용도로 만든 간단한 todo list 입니다.

## 사용 기술

- React
- Redux (redux-toolkit)
- Typescript
- styled-components
- json-server

## 사용해보기

> 로컬에서 사용해보고 싶다면 다음과 같이 실행해보세요.

```bash
yarn # or npm install
yarn dev # 개발 서버 실행
yarn json-server # json-server 실행
```

## 기능

> 현재 구현된 기능들입니다.

- [x] todo CRUD
- [x] todo 카테고리 CRUD
- [x] todo 카테고리별 todo 보기
